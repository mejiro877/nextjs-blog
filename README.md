# NextJS ブログ

![2023-07-11_17h49_37](https://github.com/mejiro877/nextjs-blog/assets/30212292/aab264f4-0990-4b37-ab42-60ba853558b0)

## サービス概要

Next.js を使ったブログサイトです。

記事一覧ページと記事詳細ページで構成されています。

記事はマークダウン形式のファイルから動的にページ生成しています。

## 実装機能

- ブログ記事の閲覧
- 記事詳細ページの動的生成

## 技術スタック

- Next.js
- React
- Vercel
